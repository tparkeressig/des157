Needfinding with Party Mode by preziotte.

  • Three-dimensional elements need to be rotatable
  • Tooltips need to be in an immediately visible/accessible place
  • “Drawers” which automatically close also need to automatically open
  • Need to be able to fast forward within a song, not just skip from one song to the next
  • Needs to be engaging for those with short attention spans
  • Needs to be shareable
  • Needs to load quickly
