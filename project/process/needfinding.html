Audience Needs:

  • 1. Three-dimensional elements need to be rotatable
  • 2. Tooltips need to be in an immediately visible/accessible place
  • 3. “Drawers” which automatically close also need to automatically open
  • 4. Need to be able to fast forward within a song, not just skip from one song to the next
  • 5. Needs to be engaging for those with short attention spans
  • 6. Needs to be shareable
  • 7. Needs to load quickly
